
install instructions: open a terminal in the 'blog' folder. run 'npm install' to install dependencies.
launch instructions: in the terminal open in the 'blog' folder, run 'npm start' to start the app.
testing instructions: an instance of the blog should open automatically in your browser. the link will also print in console.
to close instance, run 'ctrl + c' in console,  then type 'y'.
